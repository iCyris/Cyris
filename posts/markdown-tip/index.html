利用 HTML 在 Markdown 中打出数学符号

Posted on Mar 28, 2018

想要打出奇奇怪怪的符号 :D

1. 上标

n2=n+1

n<sup>2</sup>=n+1

2. 下标

a=log2b

a=log<sub>2</sub>b

3. 注册商标

hexo ®

hexo &reg;

4. function 符号

ƒ(x)=x+1

&fnof;(x)=x+1

5. 根号

√2 （感觉不是很好看）

&radic;2

6. 角度符号

30°

30&deg;

7. 一点小工作

整理了一些比较常用的特殊符号的 HTML 编码：

  空格 &nbsp;
