利用 HTML 在 Markdown 中打出数学符号

Cyris Mar 28, 2018
  • Notebook
  • Share to QQ

想要打出奇奇怪怪的符号。

1. 上标

n2=n+1

n<sup>2</sup>=n+1

2. 下标

a=log2b

a=log<sub>2</sub>b

3. 注册商标

hexo ®

hexo &reg;

4. function 符号

ƒ(x)=x+1

&fnof;(x)=x+1

5. 根号

√2 （感觉不是很好看）

&radic;2

6. 角度符号

30°

30&deg;

7. 一点小工作

整理了一些比较常用的特殊符号的 HTML 编码：

符号 说明 编码
× 乘号 &times;
÷ 除号 &divide;
← 向左箭头 &larr;
→ 向右箭头 &rarr;
↑ 向上箭头 &uarr;
↓ 向下箭头 &darr;
↔ 双向箭头 &harr;
⇐ 双线向左箭头 &lArr;
⇒ 双线向右箭头 &rArr;
⇔ 双线双向箭头 &hArr;
  空格 &nbsp;
± 正负符号 &plusmn;
∞ 无限大符号 &infin;
∠ 角度符号 &ang;
∫ 微积分符号 &int;
¿ 倒问号 &iquest;
≠ 不等于符号 &ne;
≡ 相等符号 &equiv;
≤ 小于等于符号 &le;
≥ 大于等于符号 &ge;
¶ 段落符号 &para;
§ 章节符号 &sect;
⊥ 垂直符号 &perp;
© 版权所有符号 &copy;
™ 商标符号 &trade;
